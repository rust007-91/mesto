# Проект: Место

---
Сервис Mesto: интерактивная страница, куда можно добавлять
фотографии, удалять их и ставить лайки.

В проекте применены технологии:
* HTML
* CSS
* БЭМ
* CSS Grid Layout
* CSS Flexbox
* Adaptive layout
* Работа с DOM
* JavaScript (интерактивность карточек, форм, попапов)
